Submissions

This webpage provides a service for submitting SampleTab format files as used by the BioSample Database.

Please provide the SampleTab file you wish to submit into the form below. You will then be able to download a version of that file that has been corrected and accessioned.

If you have any questions about this service, please contact biosamples@ebi.ac.uk and attach the SampleTab file you were trying to submit.

If you need an API key please contact biosamples-submission@ebi.ac.uk
